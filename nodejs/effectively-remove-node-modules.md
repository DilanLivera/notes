# Effectively Remove Node Modules

- Install [rimraf](https://www.npmjs.com/package/rimraf) package using `npm install rimraf -g` command

- Run `rimraf [path-to-the-nodemodules-folder]` command to remove

  ```powershell
  rimraf node_modules

  rimraf .\**\node_modules
  ```
